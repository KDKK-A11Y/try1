# 智能语音交互控制系统

可视化智能语音交互控制系统，支持语音和手势双模式控制。

## 🚀 功能特性

- **语音控制**: 支持中文语音指令
- **手势控制**: 使用摄像头实时识别手势，不同手指控制不同设备
- **设备管理**: 支持5种智能设备控制
- **可视化界面**: 深色主题，带发光效果的状态指示灯
- **实时日志**: 显示所有操作记录

## 🎯 手势控制说明

系统使用摄像头实时识别手势，**不同手指数量控制不同设备**：

| 手势 | 手指数量 | 控制设备 | 功能 |
|------|----------|----------|------|
| 👍 | 1指（食指） | 灯 | 切换开关状态 |
| ✌️ | 2指（食指+中指） | 空调 | 切换开关状态 |
| 🤟 | 3指（食指+中指+无名指） | 风扇 | 切换开关状态 |
| 🖖 | 4指（食指+中指+无名指+小指） | 电视 | 切换开关状态 |
| 🖐️ | 5指（手掌） | 窗帘 | 切换开关状态 |
| ✊ | 拳头 | 所有设备 | 关闭所有设备 |

### 使用示例

- **控制灯**: 伸出食指 → 灯开启/关闭
- **控制空调**: 伸出食指和中指 → 空调开启/关闭
- **控制风扇**: 伸出食指、中指、无名指 → 风扇开启/关闭
- **控制电视**: 伸出四根手指（不含拇指） → 电视开启/关闭
- **控制窗帘**: 伸出手掌（五指张开） → 窗帘开启/关闭
- **关闭所有**: 握紧拳头 → 所有设备关闭

### 手势控制流程

1. 确保摄像头已正确连接
2. 运行程序后，手势识别会自动启动
3. 将手放在摄像头前，保持手势约1秒
4. 系统识别到稳定手势后会执行相应设备的开关切换

## 🎤 语音控制说明

支持以下中文语音指令：

| 设备 | 开启指令 | 关闭指令 |
|------|---------|---------|
| 灯 | 打开灯、开灯 | 关闭灯、关灯 |
| 空调 | 打开空调、开空调 | 关闭空调、关空调 |
| 风扇 | 打开风扇、开风扇 | 关闭风扇、关风扇 |
| 电视 | 打开电视、开电视 | 关闭电视、关电视 |
| 窗帘 | 打开窗帘、开窗帘 | 关闭窗帘、关窗帘 |

### 使用示例

- "打开灯" - 开启灯光设备
- "关闭空调" - 关闭空调设备
- "开风扇" - 开启风扇设备

## 🛠️ 安装与运行

### 环境要求

- Python 3.8+
- Anaconda 或标准 Python 环境
- 麦克风（语音识别）
- 摄像头（手势识别）

### 安装依赖

```bash
pip install PyQt5 SpeechRecognition mediapipe pyaudio numpy opencv-python
```

### 运行程序

**方法1：双击启动脚本**
```
双击 start_full.bat
```

**方法2：命令行运行**
```bash
python main.py
```

### 常见问题

**问题1：pyaudio安装失败**
```bash
conda install -c anaconda pyaudio
```

**问题2：mediapipe安装失败**
```bash
pip install numpy==1.26.4 mediapipe
```

**问题3：摄像头无法打开**
- 确保摄像头未被其他程序占用
- 检查摄像头驱动是否正常

## 📝 技术实现

- **GUI框架**: PyQt5
- **语音识别**: Google Speech Recognition API
- **手势识别**: MediaPipe Hands
- **图像处理**: OpenCV
- **并发处理**: Qt QThread
